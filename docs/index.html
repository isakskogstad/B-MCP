B-MCP

Bolagsverket MCP Server - Hämta och analysera svenska företagsdata

16 Verktyg 4 Resources 4 Prompts Python 3.11+ FastMCP

Funktioner

Grundläggande företagsinfo

Hämta namn, organisationsnummer, bolagsform, status och registreringsdatum för alla svenska företag.

Finansiella nyckeltal

Analysera omsättning, resultat, soliditet, kassalikviditet och andra nyckeltal från årsredovisningar.

Styrelse & Ledning

Se styrelseledamöter, VD, revisorer och firmatecknare med roller och mandatperioder.

Riskanalys

Automatisk riskbedömning med varningar för dålig ekonomi, uteblivna årsredovisningar och obestånd.

Flerårsöversikt

Jämför nyckeltal över 4 år för att identifiera trender och förändringar.

Export

Exportera till Excel, PDF, Word, PowerPoint, CSV och JSON för vidare analys.

Installation

1

Klona repot

git clone https://github.com/isakskogstad/B-MCP.git
cd B-MCP
2

Installera dependencies

pip install -r requirements.txt
3

Konfigurera credentials

Skapa konto på portal.api.bolagsverket.se och sätt miljövariabler:

export BOLAGSVERKET_CLIENT_ID="din-client-id"
export BOLAGSVERKET_CLIENT_SECRET="din-client-secret"
4

Starta MCP-servern

python bolagsverket_mcp.py
5

Konfigurera Claude Desktop

Lägg till i claude_desktop_config.json:

{
  "mcpServers": {
    "bolagsverket": {
      "command": "python",
      "args": ["/path/to/B-MCP/bolagsverket_mcp.py"],
      "env": {
        "BOLAGSVERKET_CLIENT_ID": "din-client-id",
        "BOLAGSVERKET_CLIENT_SECRET": "din-client-secret"
      }
    }
  }
}

Säkerhet

Inga credentials i kod

Alla API-credentials läses från miljövariabler. Inget känsligt läcker till Git.

OAuth2-autentisering

Använder OAuth2 Client Credentials Flow med automatisk token-förnyelse.

.gitignore

.env-filer och credentials ignoreras automatiskt av Git.

Validering

Servern startar inte utan giltiga credentials - ingen risk för okrypterade anrop.
